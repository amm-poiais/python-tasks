1)Программа вызывается с входным параметром - путём к директории. Из данной директории необходимо удалить все файлы, идентичные по содержанию. Для сравнения содержимого необходимо использовать хэш(например md5)
2)Программа вызывается с входным параметром - путём к директории. Из данной директории необходимо удалить все пустые поддиректории.
3)Программа вызывается с входными параметрами - путём к директории1 и директории2. В директории2 нужно сохранить все файлы из директории1, сгруппированные по расширению.
4)Программа вызывается с входными параметрами - путём к директории1 и директории2. В директории2 нужно сохранить все файлы из директории1, сгруппированные по дате создания(последние сутки/последняя неделя/...).
5)Программа вызывается с входным параметром - путём к директории. Из директории необходимо удалить все ссылки.
6)Программа вызывается с входным параметром - путём к директории. Из директории необходимо удалить все файлы старше года.
7)Программа вызывается с входным параметром - путём к директории. Необходимо протестировать на целостность все архивы, содержащиеся в директории.
8)Программа вызывается с входным параметром - путём к директории. Необходимо вывести список всех файлов с совпадающими названиями.
